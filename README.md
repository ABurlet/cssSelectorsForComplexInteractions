# cssSelectorsForComplexInteractions


Challenge 1 Form Authentication

Goal:
Test the login flow by entering credentials, submitting the form, and verifying the secure area.

Skills Practiced:
Selecting input fields by ID
Typing values
Clicking buttons
Verifying URL changes
Validating success messages


Challenge 2 Hover Over

Goal:
Hover over each user avatar and confirm that the correct username becomes visible.

Skills Practiced:
Triggering hover actions
Working with .each() loops
Extracting dynamic text
Verifying hidden elements that appear on hover


Challenge 3 Dynamic Controls

Goal:
Test UI behavior when elements are dynamically added/removed or enabled/disabled.

Skills Practiced:
Waiting for elements to appear/disappear
Using { timeout: ___ } for dynamic content
Clicking buttons that trigger DOM changes
Verifying success messages
Typing into newly enabled input fields


Challenge 4 JavaScript Alerts

Goal:
Trigger and validate all types of JavaScript dialogs:
Alerts
Confirm (OK and Cancel)
Prompt (custom text)

Skills Practiced:
Stubbing window.alert, window.confirm, and window.prompt
Returning values to simulate user actions
Verifying page results
Handling browser-level modal behavior


Challenge 5 Drag and Drop

Goal:
Drag Column A onto Column B and verify the columns swap positions.

Skills Practiced:
Using DataTransfer() to simulate drag-and-drop
Triggering dragstart, drop, and dragend events
Verifying DOM changes


Challenge 6 Floating Menu

Goal:
Scroll down the page and confirm the floating navigation menu remains visible.

Skills Practiced:
Using cy.scrollTo()
Testing fixed-position UI elements
Clicking menu links and verifying anchor navigation




Through these challenges, I gained experience with
-Writing Cypress automation tests from scratch
-Navigating pages and interacting with UI elements
-Handling asynchronous behavior and dynamic loading
-Working with JavaScript alerts, confirms, and prompts
-Debugging failing tests and adjusting selectors
-Understanding Cypress events, stubs, and intercepts
-Structuring tests clearly for real QA automation work
